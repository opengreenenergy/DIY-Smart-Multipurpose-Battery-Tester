# DIY Smart Multipurpose Battery Tester

A compact DIY battery tester designed to **charge, discharge, and analyze rechargeable batteries** while measuring key parameters such as **voltage, current, capacity, and internal resistance**.

This project helps evaluate **battery health and performance** and is useful for testing lithium cells and other rechargeable batteries commonly used in DIY electronics projects.

## Key Features

• Battery **Charge Mode**  
• Constant current **Discharge Mode**  
• **Analyze Mode** for automatic charge and discharge cycle  
• **Internal Resistance (IR) Measurement**  
• Real time display of **Voltage, Current, and Capacity**  
• Simple **button based user interface**

---

## Operation Modes

**Charge Mode**  
Charges the battery until the predefined voltage limit is reached.

**Discharge Mode**  
Discharges the battery using a constant current while measuring battery capacity.

**Analyze Mode**  
Automatically performs a full charge followed by a discharge cycle to determine the **true battery capacity**.

**IR Test Mode**  
Measures the internal resistance of the battery to evaluate battery condition.
